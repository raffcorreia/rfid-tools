# Protocol Scripts

This directory is reserved for project-owned helper scripts.

Allowed examples:

- checksum verification helpers
- fixture generation helpers
- captured-frame inspection helpers

Not allowed:

- vendor SDK executables
- vendor demo scripts
- copied driver installers

## Current Scripts

| Script | Purpose |
|---|---|
| `run-yrm100-driver-tests.sh` | Builds and runs the host-side YRM100 protocol driver test binary |
